NSURLRequest/URLRequest

NSURLRequest 封装了请求的两个基本属性：要加载的URL和用于加载该请求的策略。通常用于 HTTP 和 HTTPS 请求，包括 HTTP 方法（GET，POST等）和 HTTP 标头。可变子类型是 NSMutableURLRequest 。
